# Screenshot Guide

Use this folder to store portfolio screenshots of the dashboard.

Suggested screenshots:

1. `dashboard-overview.png`
   - Capture the top of the dashboard with risk level, risk score, and total alerts.

2. `dashboard-learning-mode.png`
   - Capture the Learning Mode, OWASP Top 10 Learning Map, or Security Controls Matrix sections.

3. `dashboard-alerts.png`
   - Capture the alert table showing severity, MITRE ATT&CK mapping, and alert meanings.

4. `dashboard-response-workflow.png`
   - Capture Case Status, High Threat Actions, Countermeasure Planner, or Incident Response Summary.

These screenshots can be added to a GitHub README, portfolio page, or job application project summary.
