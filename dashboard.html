Cybersecurity Concepts Covered

OWASP Top 10 awareness This project is not a web vulnerability scanner, but login failures and suspicious access relate to application security topics like authentication, access control, and security logging.
Firewall thinking Port scan alerts help you think like a defender: identify source IPs, review whether traffic is expected, and decide what should be allowed or blocked.
Cryptography basics The project does not encrypt data yet, but it highlights why passwords, successful logins, and sensitive logs should be protected with strong hashing, encryption, and access control.
CIA triad Failed logins affect confidentiality, account lockouts affect availability, and log review helps protect integrity by checking whether activity looks trustworthy.
MITRE ATT&CK mapping Alerts are mapped to beginner-friendly ATT&CK techniques so the findings connect to common security language used by analysts.
Incident response basics The checklist, case status, notes, and countermeasure log show how an analyst can triage, document, and follow up on suspicious activity.

Learning Mode

FAILED_LOGIN What happened: one user failed to log in several times.
Why it matters: this can be password guessing or a user who forgot a password.
Concept: OWASP authentication and MITRE brute force.
Analyst check: confirm whether the user and source IP are expected.
PORT_SCAN What happened: a source IP checked network targets for open ports.
Why it matters: scanning can happen before an attacker tries a service.
Concept: firewalls, network discovery, attack surface, and internal vs external IP review.
Analyst check: decide whether the source is internal, expected, allowed, reviewed, or blocked.
ACCOUNT_LOCKED What happened: an account was locked after suspicious login behavior.
Why it matters: this can protect confidentiality but may affect availability.
Concept: CIA triad and account security controls.
Analyst check: confirm whether the lockout was expected.
SUCCESS_AFTER_FAILURE What happened: a login succeeded after repeated failures.
Why it matters: the password may have been guessed or the account may need review.
Concept: valid accounts and access control.
Analyst check: compare the login time and IP address with normal behavior.
