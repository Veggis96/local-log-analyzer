Learning Notes

Mini SIEM: raw log lines are normalized into searchable events with timestamp, event type, user, IP, and data source.
OWASP: repeated login failures connect to authentication and access-control concepts.
MITRE ATT&CK: failed logins map to Brute Force, and suspicious successful logins map to Valid Accounts.
Firewall thinking: port scans should be reviewed by source IP before any allow or block decision.
Cryptography: logs should not contain plaintext passwords; real systems should use strong hashing and encryption.
